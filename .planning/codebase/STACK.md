# Tech Stack

Core technologies used in CourtMastr v2.

## Core
- **Frontend**: Vue 3 (Composition API, `<script setup>`)
- **Language**: TypeScript (Strict mode)
- **Build Tool**: Vite
- **UI Framework**: Vuetify 3 (Material Design)
- **State Management**: Pinia
- **Styling**: Sass/SCSS, Vanilla CSS
- **Package Manager**: npm

## Backend (Firebase)
- **Database**: Firestore (NoSQL, persistent local cache enabled)
- **Authentication**: Firebase Auth
- **Serverless**: Cloud Functions
- **Hosting**: Firebase Hosting
- **Storage**: Firebase Storage

## PWA
- **Plugin**: `vite-plugin-pwa`
- **Features**: Service workers, offline support, installable icons.

## Utilities
- **Date Handling**: `date-fns`
- **QR Codes**: `qrcode`
- **Excel/Data**: `xlsx`
- **Draggable UI**: `vuedraggable`
- **Screenshot/Canvas**: `html2canvas`

## Domain Specific
- **Bracket Management**: `brackets-manager`, `brackets-viewer`, `brackets-memory-db`
