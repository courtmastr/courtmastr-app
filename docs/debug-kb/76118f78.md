## Attempts

**Date**: 2026-04-01
**Change**: Added a unit test that called `restorePreReleaseGeneratedFiles()` with no arguments to cover the destructuring-default bug path.

**Result**: ❌ `test:log` failed because that test executed a real `git restore`, which conflicted with Git's lock usage inside the logged test process.

## Root Cause
