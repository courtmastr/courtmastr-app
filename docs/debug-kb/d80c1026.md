## Root Cause
Fresh worktrees do not automatically receive the ignored production Firebase web env file, so the mandatory build guard fails before TypeScript or Vite can run.

## Fix (final)
Provide `.env.production` in the worktree (or export the required `VITE_FIREBASE_*` variables), then rerun:
```bash
cp /Users/ramc/Documents/Code/courtmaster-v2/.env.production /Users/ramc/Documents/Code/courtmaster-v2/.worktrees/<branch>/.env.production
npm run check:firebase-env
npm run build
npm run build:log
```
