# KB Entry: build:log fails when build and build:log race on dist cleanup

## Root Cause
Running `npm run build` and `npm run build:log` at the same time lets two Vite processes fight over deleting and recreating the same `dist/` tree.

## Fix (final)
```bash
# Do not run these in parallel in the same worktree.
npm run build
npm run build:log
```
